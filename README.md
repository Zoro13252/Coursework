
## 🎯 Использование

### Конвертер валют
1. Введите сумму
2. Выберите исходную и целевую валюту
3. Нажмите "Конвертировать" или введите сумму

### Бюджет поездки
1. Введите название расхода
2. Укажите сумму и валюту
3. Выберите категорию
4. Нажмите "Добавить расход"

### Сохранение данных
- Бюджет автоматически сохраняется в localStorage
- Данные сохраняются при закрытии браузера
