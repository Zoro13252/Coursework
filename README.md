# Калькулятор для путешественника

Удобный веб-калькулятор для путешественников: конвертация валют, единиц измерения и планирование бюджета поездки.

## ✨ Возможности

- **Конвертер валют** — работает с актуальными курсами ЦБ РФ (USD, EUR, RUB, GBP, JPY, CNY)
- **Конвертер единиц** — расстояние (км ↔ мили) и масса (кг ↔ фунты)
- **Планировщик бюджета** — добавление расходов в разных валютах, автоматический перевод в рубли, итоговая сумма
- Сохранение бюджета в браузере (localStorage)
- Приятный тёмный дизайн, адаптивная вёрстка

## 🚀 Как запустить

1. Скачай все файлы проекта:
   - `index.html`
   - `style.css`
   - `script.js`

2. Помести их в одну папку.

3. Открой файл **`index.html`** в любом современном браузере (Chrome, Edge, Firefox, Safari).

Готово! Калькулятор работает полностью оффлайн (кроме курсов валют).

> **Примечание**: Курсы валют загружаются автоматически с сайта ЦБ РФ при открытии страницы.

## Функционал

### Конвертер валют

- Ввод суммы и выбор валют
- Автоматическая конвертация при изменении данных
- Отображение актуальных курсов к рублю

### Конвертер единиц

- Расстояние: километры ↔ мили
- Масса: килограммы ↔ фунты
- Мгновенная конвертация при вводе

### Бюджет поездки

- Добавление расходов с названием, суммой, валютой и категорией
- Автоматический перевод всех расходов в рубли
- Удаление отдельных расходов
- Сохранение и очистка бюджета
